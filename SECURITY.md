# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do NOT create a public issue. Instead, report it by sending an email or contact the maintainer directly.

### 脆弱性の報告について

本プロジェクトでセキュリティ脆弱性を発見した場合は、パブリックな Issue を作成せず、まずはメール等でメンテナへ直接ご連絡ください。
