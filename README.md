# Multi Image Canvas

ゲーム・クリエイティブ作業中に参考画像をオーバーレイ表示するための Windows ネイティブ画像ビュアー。
.NET 8 / WinForms 製。旧 `NativeImageCanvas`（単一Program.cs構成）の全面リファクタ版。

## 主な機能

### ビュアーモード
- エクスプローラーから画像を開くと、**画像をウィンドウいっぱいに表示する閲覧専用UI**で起動
  （ツリー・オーバーレイ等の編集UIは非表示。パン/ズームのみ）
- 上部の **「キャンバスにて編集」** ボタンで通常の編集画面に切替。前回セッションのタブを
  復元した上で、この画像が載った新しいキャンバスタブが追加される
- ビュアーとして開いている間はセッションを一切読み書きしない（作業中データを壊さない）

### キャンバス共有（プライバシー重視・オフライン）
- ファイル → **共有用にエクスポート...** で画像同梱の `.mics` を作成（受け取った人はこのアプリでそのまま開ける）
- 既定でプライバシー保護がすべてON:
  - **メタデータ除去**（EXIF・GPS位置情報・撮影機材 → ピクセルから再エンコード）
  - **トリミングで隠した部分のピクセルを含めない**
  - **非表示レイヤーを除外**
  - ファイル名・フォルダ・PC名は含めない（連番に匿名化、ウィンドウ位置等の環境情報も除外）
- ネットワーク送信は一切なし（ローカルにファイルを書き出すだけ）
- 読み込み側も共有ファイルを信頼しない前提で防御: Zip Slip拒否 / エントリ数・展開サイズ上限（zip爆弾対策・実測バイト数で強制）/ 画像以外は展開しない

### インストーラー / ファイル関連付け
- `tools\make_installer.ps1` で **ユーザー単位MSI**（管理者権限不要・.NET同梱で導入不要）を作成
  - スタートメニュー登録・「既定のアプリ」への表示・アンインストールで完全撤去
- 設定 → **ファイル関連付け** タブで、このアプリで開く拡張子を個別に選んで登録/解除
  （HKCUのみ使用。既定のアプリにする最終確定はWindows設定で行う仕様のため、ワンクリックで開けるボタン付き）

## 従来からの機能

### キャンバス
- 複数画像の自由配置（無限キャンバス・固定サイズ制限なし）
- **中ボタンドラッグ / 右ボタンドラッグ(空白部) / Space+ドラッグ でパン**
- マウスホイールでカーソル位置基準ズーム（2%〜1600%）
- Ctrl+0 全体表示 / Ctrl+1 100%
- 比率固定リサイズ（角ハンドル）・辺トリミング（辺ハンドル）
- **自由回転**（上部の回転ハンドル。90°近傍スナップ、Shiftで15°刻み）
- 90°回転（Ctrl+R / Ctrl+Shift+R）・左右/上下反転（Ctrl+H / Ctrl+Shift+H)
- 画像ごとの不透明度・表示/非表示
- 他画像の端・中心への**スナップ**（Altで一時無効、編集タブでON/OFF）
- **Undo/Redo**（Ctrl+Z / Ctrl+Y、操作履歴200件）
- 複製（Ctrl+D)・削除・**一括削除**
- 右クリックメニュー（複製/削除/回転/反転/重ね順/トリミング解除）
- アニメGIF再生
- PNG書き出し（Ctrl+E、透過背景・回転反映）

### レイアウト管理
- **レイアウトタブ**: タブごとに独立したキャンバス（Ctrl+T 新規 / Ctrl+W 閉じる / Ctrl+Tab 切替）
- **レイアウト保存/読込**（.micl = JSON。Ctrl+S / Ctrl+O）
- **前回セッション自動復元**: 全タブ・ウィンドウ位置・設定を終了時に保存し次回起動時に復元
- 画像ファイルが見つからない場合はプレースホルダで配置を維持

### UI
- 上部バーはメニューバー方式（ファイル / 編集 / 表示 / オーバーレイ）
  - 一度クリックで開けば、カーソル移動だけで隣のメニューに切替（MenuStrip標準動作）
  - 通常時は枠なし・ホバー時のみ角丸で強調。ボタン類は全体的に角丸デザインで統一
- タブバー右端にズーム操作（全体表示 / − / ＋ / 倍率表示）
- アクティブタブ内・名前の左の 📝 で**レイアウト名を変更**
- **タブのドラッグ&ドロップで並べ替え**（Ctrl+←/→ の切替順にも反映）
- **Ctrl+← / Ctrl+→ でレイアウトタブ切替**（設定でキー変更可。Ctrl+Tabも使用可）
- グリッド線は常時表示（ワールド座標固定）。**グリッドスナップ**は設定でON/OFF（初期OFF）
- **オーバーレイ登場アニメーション**を設定で選択（ブロック / フェード / スライド / ワイプ / なし）
- **設定ダイアログ**（ファイル → 設定）: UIテーマ / **ショートカットキーの変更** / セッション復元ON・OFF / 自動保存間隔
- **UIテーマ**: ダーク / ライト / ミッドナイト / ハイコントラスト
- 右パネル切替: **フォルダツリー / サムネイル一覧（3列グリッド） / レイヤーパネル**
- **選択画像の操作メニュー**: 画像を左クリック選択すると、右クリックメニューと同内容の
  リストが画像の横にフローティング表示（パン・ズーム・移動に追従。不透明度はレイヤーパネル）
- オーバーレイ有効化ボタン横の ⚙ で、ファイル選択画面に重ねてオーバーレイ設定パネルを表示
  （有効化 / クリック透過 / オーバーレイ透過率）
- エクスプローラー風ツリー（クイックアクセス・OneDrive・PC・ネットワーク・UNCパス）
- F11 UI非表示 / F1 ショートカット一覧（キー変更を反映して表示）

### オーバーレイ
- 最前面オーバーレイウィンドウ（**フォーカスを奪わない** = ゲーム操作を中断しない）
- クリック透過モード
- 不透明度調整
- **グローバルホットキー**（他アプリ操作中も有効）
  - `Ctrl+Alt+H` オーバーレイ表示/非表示
  - `Ctrl+Alt+T` クリック透過切替
  - `Ctrl+Alt+PgUp / PgDn` 不透明度調整

> ⚠ フルスクリーン**排他**モードのゲーム上には表示できません（OSの仕様）。
> ゲーム側を「ボーダレスウィンドウ」に設定してください。

## 対応ファイル形式

| 経路 | 形式 |
|---|---|
| GDI+ | PNG / JPEG / BMP / GIF(アニメ再生対応) / TIFF / ICO |
| WIC (OSコーデック) | **WebP** / HEIC / HEIF / AVIF / JXR / DDS ※ |

※ WebPはWindows 11標準。HEIC/AVIFは Microsoft Store の拡張機能インストール環境で表示可能。

## ビルド

必要: .NET 8 SDK

```
cd src
dotnet build                 # 開発ビルド
build_portable_exe.bat       # 配布用単一EXE (自己完結 約63MB)
dotnet publish -c Release -r win-x64 --self-contained false -p:PublishSingleFile=true
                             # ランタイム依存版 (約0.3MB、要 .NET 8 Desktop Runtime)
```

## 構成

```
src/
  Program.cs        エントリポイント
  MainForm.cs       メインウィンドウ (リボン/タブ/セッション/ホットキー)
  MainForm.Tree.cs  エクスプローラーツリー・パス欄
  CanvasSurface.cs  キャンバス描画・入力 (パン/ズーム/変形/スナップ)
  CanvasModel.cs    ドキュメントモデル・レイアウトJSON入出力
  UndoRedo.cs       コマンドパターンUndo/Redo
  ImageDecoder.cs   GDI+/WICデコード・サムネイル・プレースホルダ
  OverlayForm.cs    オーバーレイウィンドウ
  SidePanels.cs     レイヤーパネル・サムネイルビュー
  Theme.cs          テーマ定義
  UiControls.cs     カスタムコントロール・ヘルプ
  KnownFolders.cs   既知フォルダ取得
  SessionStore.cs   セッション永続化 (%APPDATA%\MultiImageCanvas)
tests/              ユニットテスト
tools/              検証スクリプト
```

## データ保存先

- セッション: `%APPDATA%\MultiImageCanvas\session.json`（終了時＋自動保存。間隔は設定で変更可）
  - 環境変数 `MIC_SESSION_DIR` で保存先を上書き可能（ポータブル運用・テスト隔離用）
- キー割り当ての変更・テーマ・各種設定もセッションに保存
- レイアウトファイル: 任意の場所に `.micl`（JSON）
